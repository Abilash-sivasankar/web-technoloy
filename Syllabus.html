Web Technology 03 - Syllabus

Detailed Syllabus

Module I – INTRODUCTION

XHTML: Evolution of HTML and XHTML - Standard XHTML Document Structure - Basic Text Markup - Images - Hypertext Links - Lists - Tables - Forms - Frames.
Cascading Style Sheets: Introduction to CSS – Levels of Style Sheets – Style Specification Formats – Selector Forms – Property Value Forms – Font Properties – List Properties – Color – Alignment of Text – Background Images – Span and Div Tags.

Module II – XML

Introduction to SGML – Features of XML – XML as a Subset of SGML – XML Vs HTML – Views of an XML Document – Syntax of XML – XML Document Structure – Namespaces – XML Schemas – Simple XML Documents – Different Forms of Markup in XML – Document Type Declarations – Creating XML DTDs – Displaying XML Data in HTML Browser.

Module III – PERL

Origin and Use of Perl – Scalars and Their Operations – Assignment Statements – Simple Input and Output – Control Statements – Fundamentals of Arrays – Hashes – Functions – Pattern Matching – Simple Programs in Perl – Using Perl for CGI Programming.

Module IV – PHP

Origin and Use of PHP – Overview – General Syntactic Characteristics – Operations and Expressions – Control Statements – Arrays – Functions – Pattern Matching – Form Handling – Files – Cookies – Session Tracking.

Module V – MySQL

Basics – Query Design and Functions – Database Operations – Procedures – Simple Programs in PHP and MySQL.
